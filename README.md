# Hemanth Kumar Portfolio Website

A modern, responsive personal portfolio website showcasing Hemanth Kumar's AI & Automation expertise. Built with HTML, CSS, and JavaScript.

## Features
- Fully responsive design (desktop, tablet, mobile)
- Interactive sidebar with contact details
- Project showcase with filterable categories (AI Chatbots, Automation, Content Creation)
- Blog section with AI and automation articles
- Testimonials modal

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari)

### Running the site
1. Open `index.html` directly in your browser, **or**
2. Serve the folder with any static HTTP server (e.g., `npx -y serve .`).

## Usage
Navigate through the sections:
- **About** – Overview of Hemanth's background.
- **Services** – AI Prompt Engineering, Chatbots, Content Creation, CRM Management.
- **Portfolio** – Filterable project gallery.
- **Blog** – Articles on AI automation and content creation.
- **Contact** – Email and phone details.
